What is Xaringan?

  • Xaringan is a R package for creating slideshows through R Markdown.



  • You can use R Markdown through RStudio and it is easy to get started too.



  • All you need is go from the menu File -> New File -> R Markdown -> From Template -> Ninja Presentation, then you will be able to view it by pressing the Knit button.
